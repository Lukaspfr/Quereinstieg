# Windows Bootcamp (im Aufbau)

Dieser Bereich dokumentiert meinen strukturierten Aufbau von
Windows Server / Client Kompetenzen mit Fokus auf System Administration.

## Ziel

- Solides Windows-Admin-Fundament
- PowerShell-Grundlagen
- Windows Services & Eventlogs verstehen
- Netzwerk-Diagnose unter Windows
- Active Directory Grundkonzepte (Users, Groups, OU, GPO Basics)

Langfristiges Ziel: System Engineer mit starkem Infrastruktur- und Plattform-Fokus.
